## Bash-Scape
```
/bin/bash -> bash
${!#}<<<
{
  $\'
  -> l
  \\${##}
  $((
    $((
      ${##}<<${##}<<${##}
    ))
    #${##}${##}
  ))
  $((
    ${##}<<${##}<<${##}
  ))

  -> s
  \\${##}
  $((
    $((
      ${##}<<${##}
    ))
    #${##}${##}$#
  ))
  $((
    $((
      ${##}<<${##}
    ))
    #${##}${##}
  ))
  \'
  -> ejecuta comando
  ,
}
```

Formas de ejecutar (ls) Octal:
```
1 - /bin/bash -c ls
2 - ${!#}<<<{$\'ls\',}
3 - $'\154\163' -> Octal
4 - ${!#}<<<$\'\\154\\163\'
5 - ${!#}<<<{$\'\\154\\163\',}
6 - ${!#}<<<{$\'\\${##}$(($((${##}<<${##}<<${##}))#${##}${##}))$((${##}<<${##}<<${##}))\\${##}$(($((${##}<<${##}))#${##}${##}$#))$(($((${##}<<${##}))#${##}${##}))\',}
```
